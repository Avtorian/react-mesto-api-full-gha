# Проект: Mesto-react-auth
Проект по созданию интерактивной страницы с возможностью регистрации и авторизации, куда можно добавлять фотографии, удалять их и ставить лайки.
### Технологии
* HTML
* CSS
* Figma
* JS
* React
* Rest Api
